# Utility Billing Interface (UBI)

Tools for generating Entrata Core formatted Excel uploads and invoice PDFs for UBI.

## Contents

- `Generate Entrata Core Formatted Excel Upload Documents of UBI Transactions.py` — main generator (exe provided).
- `Generate Entrata Core Formatted Excel Upload Documents of UBI Transactions.exe` — packaged executable.
- `Select File and Write PDF Invoice.py` — PDF invoice helper.
- `UBI_ALLOCATION.sql` — allocation SQL logic.
- `.env` — configuration (not committed).

## Usage

- Run the executable or the Python scripts after configuring `.env` as needed.
