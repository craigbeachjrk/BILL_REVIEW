DEPLOYMENT EXCLUSIONS - DO NOT MODIFY WITHOUT UNDERSTANDING

The deploy_app.ps1 script is configured to create a LEAN deployment package.

---------------------------------------------
- aws_lambdas/     (241 MB) - Lambda code deployed separately via Lambda deploy scripts
- infra/           (0.08 MB) - Infrastructure/Terraform code, not needed at runtime
- deploy/          (4.2 MB)  - Build artifacts
- test/            (0.67 MB) - Test data
- secrets/         (0.48 MB) - SECURITY RISK - never deploy secrets
- scripts/         (0.07 MB) - Dev scripts
- .claude/         (0.01 MB) - Claude config
- .git/            - Git history
- __pycache__/     - Python cache
- .venv/, venv/    - Virtual environments
- node_modules/    - Node dependencies
- dd=*/            - Debug/test data folders (pattern match)

--------------------
- *.zip            - Build artifacts (e.g., aggregator_deploy.zip at 30 MB)

-----------------------------------

- data/            - Config data files

Files:

- auth.py          - Authentication

- utils.py         - Utilities

- Dockerfile       - Container build
- buildspec.yml    - CodeBuild spec
- apprunner_config.json
- VERSION.txt
